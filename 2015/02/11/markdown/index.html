Markdown学习练习笔记/兼文档

哲学 philosophy

咳咳。简单说就是“易读”，“易写”。

Markdown is intended to be as easy-to-read and easy-to-write as is feasible.

标题 HEADERS

一级标题 H1

# 这是一个一级标题 header1

这还是一个一级标题 header1
===========

二级标题 H2

## 这是一个二级标题 H2

这还是一个二级标题 H2
--------

三级-六级标题

### 三级标题 H3
……
###### 六级标题 H6

只有一种写法了。

引用 Blockquotes

> Smile, breathe and go slowly.

Smile, breathe and go slowly.

无序列表 Unordered lists

- 第一点
- 第二点
  • 第一点
  • 第二点

3. 第一点
2. 第二点

  2. 第二点

链接

行内式

[曼珠沙华](http://blog.wileam.com/ "生活博客")

曼珠沙华

参考式

[曼珠沙华][1], [技术博客][2]
[1]: http://blog.wileam.com/ "(此处可选)生活博客"
[2]: http://code.wileam.com/ "(此处可选)技术博客"

曼珠沙华, 技术博客

注意最下面的参考链接是直接不会出现的。
